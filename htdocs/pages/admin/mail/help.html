Placeholders

The following place holders are available:

Placeholder Description
${sirname} Sirname of the user
${name} Name of the user
${activation_link} Activation link
${ce1sus_url} URL of this ce1sus
${username} Username
${password} Password in plaintext

The following place holders are available for the subject:

Placeholder Description
${event_id} Internal event identifier
${event_uuid} UUID of the event
${event_tlp} TLP level
${event_risk} Risk
${event_title} Title of the event

The following place holders are available for the body:

Placeholder Description
${event_url} Direct link to the event
${event_uuid} UUID of the event
${event_created} Creation date
${event_reporter} group name of the creator
${event_risk} Risk
${event_analysis} Analysis
${event_description} Event description
${event_relations} All the event objects and their attributes
${event_objects}
${event_updated_relations} All the links of the related event, which changed since the last publication
${event_updated_objects} All the event objects and their attributes, which changed since the last publication
