Attributes A ce1sus attribute corresponds to a cybox object property, in this section the user is allowed to add custom attributes.

NOTE: Only custom attributes can be edited or deleted. Also to note is that if an object is still referenced it cannot be deleted.


Attribute Properties
Name Description Mandatory
name Name of the attribute yes
description Description of the attribute yes
CHKSUM Checksum of the attribute, it is used to determine a definition uniquely not applicable
Regex A regular expression specifying the format of the expected value. The default value is "^.+$" yes
Data type The base data type of the attribute value. Can be (String, Date, DateTime, Integer, etc....). The default value is "String" yes
Base type The base type of the attribute value. This is used to regroup one or more attributes for the generation of relations. The default value is "None"
Example: If Attribute Domain_Value and Hostname_Value have the same Base type then a relation is created between them else only the values of their own definition is taken into account. 		yes
Default Condition The condition selected by default when adding a new attribute. The default value is "Equals" no
Relationable If checked a relation is created when matching elements are found. no
Default Shareable If checked the share flag is automatically set when this attribute is added to an object. no
